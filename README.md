# Rectangle integration

## To run

### Compile

```bash
g++ src/*.cpp src/*.h -o out/main
```

### Execute

```bash
out/main <a> <b> <n> <threads>

# `a` - нижний предел интегрирования  
# `b` - верхний предел интегрирования  
# `a` - количество прямоугольников  
# `threads` - количество потоков 
```
